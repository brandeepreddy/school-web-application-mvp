# School Web Application MVP

A responsive school management MVP with separate admin and student portals.

## Monorepo structure

- `web/` — React + Vite + TypeScript + Tailwind frontend
- `server/` — Node.js + Express + TypeScript backend
- `docs/` — schema and API documentation

## Planned MVP scope

- Authentication for admin and student users
- Admin dashboard and management modules
- Student portal with personalized data
- Attendance, marks, homework, events, notifications, reports
- Role-based access control and audit logs

## Getting started

1. Copy `.env.example` to `.env`
2. Install dependencies with `npm install`
3. Run both apps with `npm run dev`

Detailed setup and API documentation will be finalized after implementation.
